# 什么是通货膨胀？
### 定义
流通中的**货币数量超过经济实际需要**而引起的货币贬值和物价水平全面而持续的上涨--用更通俗的语言来说就是：在一段给定的时间内，给定经济体中的物价水平普遍持续增长，从而造成货币购买力的持续下降。
### 原因
货币数量超过经济实际需要。
### 后果
相同金额的金钱的购买力下降。
