# 2、副词和形容词的区别？

- 副词和形容词一样都有修饰功能。
- 形容词修饰名词，副词可以修饰动词，形容词，副词， 或整个句子。
- 副词按词汇意义可以分为方式副词，程度副词，地点副词，时间副词，频度副词，疑问副词。
- 副词也有比较等级的变化。

# 6、副词的比较等级

- 副词比较级的构成和意义类似于形容词的比较级。

- 一般可在词后加后缀-er，-est构成，还可借助more，most构成比较级和最高级。
- 少数副词的比较级是不规则的。
