# 数据协议
1. 编辑器生成json数据，小程序根据json数据展示页面
2. json数据结构
    1. json的根节点用于描述页面
        - 页面名称
        - 页面css样式
        - 页面的小程序配置
            - 导航栏颜色、字体等
            - 是否可以分享
        - 组件列表children, 每一项都是一个描述组件的对象
    2. 组件描述对象
        - 组件样式style
        - 组件属性attributes
            - 组件定义好的属性，可能是小程序原生组件，也可能是自定义组件。

    3. 自定义模块subpages
        - 一组组件的集合
        - 可先创建模块，然后再需要的地方使用
        - 主要是满足组件组合，和复用的需求。

# 编辑器
### 微页面列表
- 创建分类
- 分类下创建页面
- 操作
    - 编辑页面内容
    - 复制
    - 删除
    - 修改页面名称、分类
    - 定时发布
    - 推广
    - 设置为
        - 主页
        - 商城主页
        - ....

### 编辑微页面
- 页面大纲
    - 显示页面组件的层次结构树， 点击节点快速定位视图
- 页面模板
    - 常用的设置
- 新增组件
    - 点击或者拖拽组件到指定位置
- 页面展示区域
- 组件属性设置区域
- json查看区域

### 微页面列表
- 创建分类
- 分类下创建页面
- 操作
    - 编辑页面内容
    - 复制
    - 删除
    - 修改模块名称、分类
    - 定时发布
    - 设置为
        - 主页
        - 商城主页
        - ....
